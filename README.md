# Local Service Business Website Redesign

## Project Overview
A redesign of a local AC repair service website focused on lead generation and conversion optimization.

## Objectives
- Improve user experience
- Increase customer inquiries
- Create a mobile-friendly layout
- Optimize call-to-action placement

## Pages Designed
- Homepage
- Service Page
- Contact/Lead Page

## Tools Used
- Figma
- Canva

## Key UX Decisions
- Clear navigation
- Strong call-to-action buttons
- Trust-building testimonials
- Simplified lead form
